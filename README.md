# PETSTORE REST API

This repo is for the Petstore Rest API backend portion of our SE2 Project.

### Technologies used:

- Node.js
- Typescript
- Express
- MySQL

### How do I get set up?

1. Clone Repo
2. Run `npm i` to install all dependencies
3. Run `npm start` to start up the server - not yet implemented!!!
